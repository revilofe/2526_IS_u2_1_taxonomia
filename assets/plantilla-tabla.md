### Características del Incidente

| Campo | Contenido |
|-------|-----------|
| **Descripción** | ¿Qué es exactamente este tipo de incidente? Define claramente el concepto. |
| **Funcionamiento** | ¿Cómo funciona? Describe el proceso de ataque/ocurrencia en 3-5 líneas. |
| **Identificación** | ¿Cómo lo detecto? Lista indicadores, señales, logs o comportamientos sospechosos. |
| **Protección** | ¿Cómo me protejo? Medidas preventivas y técnicas de mitigación. |

---

### Caso Real Documentado

| Parámetro | Contenido |
|-----------|-----------|
| **URL** | Enlace a noticia/documento del incidente |
| **Breve Descripción** | Resumen de qué pasó (3-4 líneas) |
| **Agrupación/Tipo** | Clasificación según taxonomía INCIBE |
| **Origen** | Interno / Externo / Desconocido |
| **Perfiles Afectados** | ¿Qué usuarios? (ej: administrativos, dirección, operarios) |
| **Nº y Tipología Sistemas** | Cantidad y tipo (servidores, PCs, móviles, red, etc.) |
| **Categoría Sistemas** | Crítica / Alta / Media / Baja |
| **Peligrosidad** | 1 (Bajo) - 5 (Crítico) |
| **Impacto** | L1 (Bajo) - L5 (Crítico) |
| **Prioridad Asignada** | Emergencia / Alta / Normal / Baja |

---

## Instrucciones de Relleno

1. **Descripción:** Busca definiciones en los apuntes o en la matriz INCIBE-CERT
2. **Funcionamiento:** Explica de forma técnica pero comprensible
3. **Identificación:** Piensa en logs, síntomas, alertas que verías en sistemas
4. **Protección:** Medidas técnicas y organizativas
5. **Caso Real:** Busca noticias reales, incluye URL verificable

**Ejemplo de búsqueda:**
- "WannaCry ransomware 2017 caso"
- "Phishing BBVA 2022"
- "DDoS Wikipedia 2015"
- "Breach Equifax 2017"
